# 🚀 Saksham Kumar's Portfolio

A modern, responsive, full-stack personal portfolio website built to showcase my projects, skills, and professional experience. Built with the **MERN Stack** (MongoDB, Express.js, React.js, Node.js) and Tailwind CSS, this project features a seamless frontend and a robust backend with administrative capabilities.

## ✨ Features
* **Modern Frontend:** Fast and responsive user interface built with React, Vite, and Tailwind CSS. Includes dynamic animations and scrolling effects.
* **RESTful API Backend:** Scalable backend built with Node.js and Express.js.
* **Authentication:** Secure login and session management using JWT (JSON Web Tokens) and bcryptjs.
* **Project & Blog Management:** Dynamic routes to manage and display personal projects and blog posts.
* **Contact Form:** Integrated with `nodemailer` to receive messages directly from the website.
* **Document Handling:** Resume and document uploads handled securely using `multer`.
* **Database:** MongoDB via Mongoose for structured and efficient data storage.

## 🛠️ Tech Stack
**Client:**
* React (with Vite)
* Tailwind CSS
* React Router

**Server:**
* Node.js & Express.js
* MongoDB & Mongoose
* JSON Web Tokens (JWT) & bcryptjs (for Authentication)
* Nodemailer (for Emails)
* Multer (for File Uploads)

## 🏃‍♂️ Run Locally
To run this project locally, clone the repository and run the following commands:

**1. Start the Backend:**
```bash
cd server
npm install
npm run dev
```

**2. Start the Frontend:**
```bash
cd client
npm install
npm run dev
```
